Stage Thalos

Le Stage de Chez Thalos était du 17 mai au 18 juin.



Mon stage au sein de Thalos n’aurais pas été possible si Mr Normand connaissais mon grand-père, j’ai donc été le stagiaire de Sylvain Nohannic responsable des systèmes d’information au service R&D de Thalos.

En début de stage nous avons préparer deux machines serveurs pour les envoyer sur un bateau, ces machines étais des machines qui servaient à gérer les données sauvegardées par les cameras de vidéo-surveillance sur les bateaux équipés, il y a un dispositif de Redondance de stockage ( RAID 1 ) en cas de problèmes et aussi pour faciliter la récupération et archivage des vidéos.

ce système de vidéo surveillance s’appelle OCEANLIVE.




Nagios est une application libre qui permet la surveillance système et du réseau. Elle surveille des hôtes et/ou des services spécifiés, alertant lorsque les systèmes ont des dysfonctionnements et quand ils repassent en fonctionnement normal.

Par la suite j’ai travaillé sur un logiciel nommé Nagios pour étudier le Monitoring grâce au protocole SNMP ( Simple Network Management Protocol )

Début sur Nagios, j’ai tout d’abord étudier comment marchais le protocole SNMP, c’est un protocole de Couche 7 du modèle OSI, il permet la remontée d’informations via des OID


WSL ( Windows Subsystem for Linux )

Le sous-système Windows pour Linux permet aux développeurs d'exécuter un environnement GNU/Linux - y compris la plupart des outils de ligne de commande, des utilitaires et des applications - directement sur Windows, sans modification, sans la surcharge d'une machine virtuelle traditionnelle ou d'une configuration à double démarrage. Il est disponible sur Windows 10 et Windows Server 2019 en deux versions

pour mon stage j’ai travaillé avec WSL sur une machine Windows Server 2019.
