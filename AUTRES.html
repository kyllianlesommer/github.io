Scripts utile avec powershell

PowerShell est un langage de script fondé sur la programmation orientée objet.

objectif du script :

Je me suis mis comme objectif de travailler sur un script qui permet de changer une configuration ip d'une carte réseau sans passer par les nombreux labirynthes d'interface de Windows.


			
<#
-----------------------------------------------------------------------
NOM : script_changement_ip.ps1
AUTEUR : Kyllian Le Sommer
DATE : 10/11/2021
VERSION : 1.0
COMMENTS : script de changement d'adresse ip
PowerShell : Version 5.1.19041.1237 ($PSVersionTable)
Autorisation exécution script local : Set-ExecutionPolicy RemoteSigned
-----------------------------------------------------------------------
#>
$i = 0
do
{
        Write-Host "---- AFFICHAGE DES PARAMETRAGES IP DE LA MACHINE ----"
        ""
        Write-Host "-------------- INTERFACES DISPONIBLES ---------------"
        Get-NetIPInterface | Select-Object -ExpandProperty InterfaceAlias
        $int = Read-Host "entrez le nom de l'interface sans fautes"
        Write-Host "L'interface sélectionnée est $int"
        ""
        Write-Host "-------------- ADRESSES IP ACTUELLES ----------------"
        Get-NetIPAddress | Select-Object -ExpandProperty IPAddress
        $ip =          Get-NetIPAddress -InterfaceAlias wi-fi | Select-Object -ExpandProperty IPv4Address
        $newip = Read-Host "entrez la nouvelle @IP"
        Write-Host "L'interface sélectionnée est $newip"
        ""
        Write-Host "--------------- PASSERELLE ACTUELLE -----------------"
        $oldgate = Get-CimInstance -ClassName Win32_NetworkAdapterConfiguration | select-object -ExpandProperty DefaultIPGateway
        Write-Host "La passerelle Actuelle est $oldgate"
        $newgatew = Read-Host "entrez l'@IP de la passerelle"
        ""
        Write-Host "------------------- MASQUE ACTUEL -------------------"
        $oldmask = Get-NetIPAddress -IPAddress "$ip" | Select-Object -ExpandProperty prefixlength
        $oldmask
        $newmask = Read-Host "entrez le masque par défaut"
        ""

        New-NetIPAddress -InterfaceAlias $int -IPAddress $newip -DefaultGateway $newgatew -PrefixLength $newmask -Confirm -ErrorAction SilentlyContinue
        $cond = Read-Host "entrer OK pour continuer"
        if ($cond -eq "OK") {"ça continue"
        }else{Break}

}
until($i -ne 1)
write-host "Configuration Terminée"

Modification du bashrc pour des alias


dans le dossier personnel de chaque utilisateur linux nous avons un fichier qui s'apelle .bashrc qui est invisible, pour le voir il faut donc utiliser la commande "ls -a". dedans nous pouvons trouver des paramèttres du compte.

Un d'entre eux est très utile, il permet de créer des alias de commandes:

par exemple, la commande "ll" n'existe pas, elle à été crée pour éviter de taper "ls -l"

la syntaxe n'est pas compliquée, il sufit de reproduire de schema :

alias (la commande que vous voullez créer)='la commande a remplacer'


alias s='sudo'

alias shut='s shutdown -h'

alias geoip='s geoiplookup -f /usr/share/GeoIP/GeoLiteCity.dat'

alias resapache='s /etc/init.d/apache2 restart'

alias resmysql='s /etc/init.d/mysql restart'

alias .. ='cd ..'
